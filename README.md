# Liste des fonctionnalités
## Etape 1 - Facile
- Crud : create, read, update, delete
- Système de like
---
#### L'utilisateur doit au minimum pouvoir poster des images avec éventuellement un texte descriptif, avec la possibilité de liker ou disliker. Celui ayant crée le post doit pouvoir le supprimer et le modifier. Les autres utilisateur pourront seulement lire le post
---
## Etape 2 - Moyen
- Gérer la possibilité d'ajouter du code
- Système de filtre et de tri
- Gérer pagination
---
#### L'utilisateur doit pouvoir trier et filtrer les posts : par exemple par date de création. Il y aura aussi une pagination sur la quantité de post qui s'afficheront sur la page du user
---
## Etape 3 - Hardcore
- Système de gestion des users
#### L'utilisateur pourra supprimer son compte et modifier ces informations

# Route 
- Une main page avec les posts
- Une page de création et de modification de posts
- Une page de gestion des users
- Une page de d'inscription et de connexion
- Une page consacrée à la recherche et au filtre par tag
